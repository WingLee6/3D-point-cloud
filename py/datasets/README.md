# README

## 数据集说明:
本数据集包括:
1. L515
    + 来源: Intel RealSense L515相机采集的点云数据集
    + 格式: `.ply`
    + 数量: 28个
2. ModelNet40
    + 来源: ModelNet40数据集
    + 格式: `.off`
    + 数量: 
        + 40个项目(包括飞机等数据)
        + 每个项目包括测试集和训练集, 分别100余条
        
3. point_clouds_simulated
    + 来源: 模拟平面数据
    + 格式: `.ply`
    + 数量: 50个demo数据集
    + 说明: 用于测试的性能, 并非真实数据集
